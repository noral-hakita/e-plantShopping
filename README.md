# Paradise Nursery Shopping Application

This is a React-based e-commerce application for a plant nursery. It allows users to browse different categories of plants, add them to a shopping cart, and manage their orders.

## Features
- **Product Listing:** View various plants with prices and descriptions.
- **Redux State Management:** Uses Redux to track items in the cart and handle "Add to Cart" button states.
- **Shopping Cart:** View total costs, increase/decrease item quantities, and remove items.
- **Responsive Navbar:** Includes a dynamic item counter.

## How to Run
1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the local development server.
3. Run `npm run deploy` to update the GitHub Pages site.

## Tech Used
- React
- Redux Toolkit
- CSS
- Vite
